# SDM-church

SwiftUI iOS app source for the SDM youth ministry experience.

## Highlights
- SwiftUI + Observation architecture
- Shared navigation coordinator and app state injection
- Design-system driven spacing, typography, color, radius, and glass surfaces
- Dark mode, Dynamic Type, and accessibility-aware components

## Project notes
- App entry point: `App/SDMYouthApp.swift`
- Root navigation: `App/Rootview.swift`
- Design system: `DesignSystem/`
- Features: `Features/`

See `PRODUCTION_READY.md` for architecture, validation, and deployment guidance.
